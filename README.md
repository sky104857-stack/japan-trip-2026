## 每日重點

- **8/4 村上 → 米澤**：國道113號經小國町、赤芝峡；上杉神社/松が岬公園、上杉博物館；米澤牛晚餐；住宿 ホテルスロービレッジ米沢（已確認）
- **8/5 米澤 → 豬苗代 → 磐梯町**：天鏡閣、野口英世記念館、猪苗代湖畔；住宿 磐梯山温泉ホテル by 星野リゾート（已確認,不可取消/退款）
- **8/6 磐梯町 → 日光**：改為直接住日光，避開 8/7 一早開車；住宿 Fairfield by Marriott Tochigi Nikko（已確認）
- **8/7 日光 → 上野**：日光東照宮、輪王寺、二荒山神社；いろは坂→中禅寺湖→華厳の滝；還車後住宿 Hotel RESOL Ueno（已確認,不可退款）
- **8/8 上野 → 成田**：自由活動、スカイライナー、晚班機返台
